3B — Responsive improvements

Mobile & tablet fixes

Current issues

  • Sidebar occupies full width on mobile
  • Asset cards overflow on small screens
  • Header collapses awkwardly
  • No mobile sidebar toggle (hamburger menu)
  • Social media overlays don't fit mobile viewports
  • Image lightbox zoom/pan broken on touch

Fixes

  • Sidebar: overlay/drawer on mobile (<768px)
  • Asset cards: stack vertically, full-width images
  • Header: compact mode with hamburger + user pill
  • Lightbox: touch-friendly pinch-zoom, swipe close
  • Campaign grid: 1-col mobile, 2-col tablet, 3-col desktop
  • Tab buttons: full-width on mobile

3C — Accessibility & refinement

Focus & keyboard
  • Visible focus rings on all interactive elements
  • Tab order follows visual layout
  • Escape closes modals/dropdowns
  • Enter/Space activates buttons
Screen readers
  • aria-labels on icon-only buttons
  • role attributes on custom interactive elements
  • alt text on all images (use captionText)
  • aria-expanded on collapsible sections
Visual polish
  • Consistent spacing scale (4/8/12/16/24/32)
  • Subtle transitions on hover/active (150ms)
  • Card shadows unified (shadow-xs → shadow-sm on hover)
  • Status badges color-refined
  • Typography: consistent font sizes

Virtualized sidebar list
