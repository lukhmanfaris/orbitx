1utils/api.ts — apiGet/apiPost/apiPut/apiDelete wrappers. Auto-JSON parse, throw on !ok with error body. 400/500 → throw with message.
2hooks/useToast.ts — toasts[] state, addToast(type, title, msg), removeToast(id). Auto-dismiss after 5s. 4 types: success, error, warning, info.
3components/common/ToastContainer.tsx — fixed bottom-right, animated stack (Motion), auto-dismiss with progress bar.
4components/common/ErrorBoundary.tsx — class component wrapping children. Catches render errors, shows fallback UI with "Reload" button.
5components/common/Skeleton.tsx — reusable Skeleton component (width, height, rounded props). SkeletonAssetCard, SkeletonArticleRow.
6Wire into hooks: Replace all alert() and console.error() with addToast(). Wrap try/catch to show error toasts instead of silent failures. Loading skeletons in MediaTab, ArticlesTab.
